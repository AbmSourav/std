# STD
ToDo and Note taking Desktop App. 
Build with Vanilla JS, NodeJS, ElectronJS

Install on any Linux distro using **Snap**. On Ubuntu 20.4 you can also install from Ubuntu App Store.

### Snap Installation
`sudo snap install std`
